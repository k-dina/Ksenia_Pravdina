При выполнении программного кода на языке PERL:

#!/usr/local/bin/perl
print "Content‐type: text/html\n\n";
print "<H1>Hello World</H1>\n";

Программа выведет:

Hello, World!

Однако в браузере у нас будет отображаться результат выполнения этой программы:

Hello, World!
